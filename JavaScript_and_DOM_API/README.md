# Zenva - JavaScript and DOM API
My versions of the example programs from the Zenva course.

- DOM is the document object model
- Elements of a webpage in a tree-like format
- Document is the root or parent
- All other nodes are children
- Nodes are HTML elements
- Child nodes can have additional child nodes
- Javascript is one way to access the DOM
- Open up a <script> tag in html page
- Easily access and manipulate the webpage
